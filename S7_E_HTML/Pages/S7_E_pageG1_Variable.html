1.4.1.1 Variable

Une variable définit en informatique un espace dans la mémoire de l'ordinateur representant une information quelquonque. Elle peut être modifié selon des instructions.

1.4.1.2 Type d'une variable

Comme dit précédement, une variable répresente une information sous une forme de code binaire. Le type d'une variable est le format utilisée par le programme pour la traiter.

  • Les entier

    Les entiers comme nous l'indique leur nom, ils permettent de représenter des nombres entiers selon le code binaire dans l'emplacement de la mémoire

    Nombre binaire Valeur entière
    1 0 1 5
    1 1 1 1 1 1 1 1 255

    Exemple de définition de variable

    //En python

    n=int(6)

    //En c et c++

    int main()
    {
    int n = 5;
    return 0;
    }

  • Les flotant

    Les flottants eux, correspondent aux nombres à virgule. Ils sont généralement encodé dans la mémoire suivant un exposant (10,10²,e7,ect...) et un entier.

    Nombre à virgule Nombre en écriture scientifique
    0.005 5e-3
    4000000000 4e9

    Exemple de définition de variable

    //En python

    n=float(1.323219)

    //En c et c++

    int main()
    {
    float n = 1.323219f;
    return 0;
    }

  • Les booléans

    Les booléans font partie des mathématiques booléans. Leurs but est de représenter des états binaires (oui/non,vrai/faux,1/0) pour pouvoir réaliser des opérations liées à l'algorithmie. Dans la mémoire, ils correspondent généralement à un espace vide(faux) et a un espace rempli (vrai).

    //En python

    n=True

    //En c il n'y a pas de valeur "bool", soit c'est 0 (faux), soit c'est un nombre quelquonque (vraie)

    int main()
    {
    int n = 1;
    return 0;
    }
    //En c++, il y une valeur booléan mais le fonctionnement est le même
    int main()
    {
    bool n = true;
    return 0;
    }

  • Les caractères

Les caractères sont pour faire simple les lettres et elles vont composer les chaînes de caractères qui seront vu plutard. Dans la mémoire, on assigne à chaque lettre un nombre allant de 0 à 255 : c'est le code ASCII (code standard américain pour l'échange d'information).
