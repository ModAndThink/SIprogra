Les Fonction

En informatique, les fonctions sont des parties de code qui exécute des action déterminées au préalable afin de transformer des données. Les fonctions sont utilisées afin d'éviter les répétitions dans le code pour ainsi faciliter la rédaction de programme et aussi pour faciliter la lecture du code par une personne extérieure. Pour opérer, les fonction ont besoin de données en entrée puis exécute le code qui la compose et renvoie une ou plusieurs données en sortie.

Une fonction renvoie à une partie du programme qui va être exécuté directement avant de retouner à la ligne où on a fait appel à la fonction : cela permet d'exécuter un sous programme.
