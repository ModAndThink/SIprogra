La fonction "TRAITER"

Dans la chaine d’information (en orange ci-dessous), la fonction «Traiter» reçoit des signaux en provenance de la fonction « Acquérir ». Ces signaux, mis en forme ou non, sont issus de capteurs externes ou internes au système. Le traitement permettra l’analyse des informations, leur structuration et leur transmission vers la fonction « Communiquer ».

Les supports techniques

Les supports techniques câblés

Les solutions câblées sont souvent anciennes mais elles présentent l’avantage d’être robustes et ne nécessitent pas de compétence de programmation. Elles consistent à réaliser un circuit (électrique, pneumatique, hydraulique) spécifique à l’application. Il est difficile de modifier le comportement du système une fois qu’il a été élaboré. Certains composants restent paramétrables (régulateur de vitesse, temporisations, etc.).

Les supports techniques programmés

La structure d'un système programmable

Architecture basée sur un microcontrôleur

Un microcontrôleur est un composant électronique (circuit intégré) qui regroupe toutes les fonctions d’un système minimal (Processeur, RAM, ROM, et périphériques d’entrée sortie). Il effectue les opérations élémentaires qui ont été structurées dans le code représentant la suite logique des actions que doit mener le système (Programme = code machine). Le code est stocké dans la mémoire morte. La communication avec l’extérieur s’effectue par les bus d’entrée et de sortie. D’autres bus sont disponibles en fonction des applications souhaitées. Les bus transportent l’information. Une mémoire se comporte comme un meuble à tiroir : les informations stockées dans la mémoire sont appelées « données », et chaque tiroir possède une « adresse ».


Architecture basée sur un système microprocesseur

Un soc (System on chip) contient généralement un processeur assez puissant tournant à plusieurs centaines de mégahertz et reposant sur plusieurs dizaines voire centaines de mégaoctets de mémoire vive. Cette technologie est plus consommatrice d’énergie, mais est plus riche et plus puissante que les technologies basées sur microcontrôleur.


Vocabulaire

La mémoire morte (R.O.M) : Les données contenues dans la R.O.M (Read Only Memory) ne peuvent être effacées ni lors du fonctionnement du système, ni lorsqu’il est mis hors tension. Elle est utilisée pour stocker le code qui sera exécuté par le microprocesseur.

La mémoire vive (R.A.M) : Les données contenues dans la R.A.M (Ramdom Acces Memory) sont perdues lorsqu’elle est mise hors tension. Le microprocesseur utilise cette mémoire lors de l’exécution du programme.

Les bus analogiques : Ils permettent au système de traduire une grandeur électrique qui représente une grandeur physique. Le système dispose de convertisseurs analogiques numériques (CAN) qui convertissent la grandeur analogique en grandeur logique. En sortie il est possible de convertir le signal numérique en signal analogique par l’intermédiaire de convertisseurs numériques analogiques (CNA).

Les bus logiques : Ils fonctionnent en tout ou rien (TOR). Une entrée ou une sortie logique peut prendre deux valeurs : 0 ou 1, qui correspondent généralement aux tensions 0V et 5V.
