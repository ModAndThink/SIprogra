1.5 Langage informatique

Les languages informatiques existent depuis les tous premiers ordinateurs. Ce qui différencie chaque langage est en premier lieu leur synthaxe (les séparations d'actions, les fonctions, ect...). Cependant la vraie différence viens de leur compilation, les langages peuvent proposer des fonctions de base et le programme compilé peut, selon notre utilisation du langage, être plus au moins optimisé. Ainsi les langages peuvent se spécialiser dans des domaines précis. On utilise par exemple les langages c (c, c++ et c#) pour des applications sur ordinateur ou l'assembleur pour des systémes embarqués.

Les types de langages

Les langages peuvent être divisés en deux catégories : les langage structurés et les langages objets. Leur différence vient de la manière dont un script est construit. En langage structurés, les actions s'executent les une derrière les autres dans un seul script.
Dans le langage objet, le programme est fragmenté en objet qui sont des structures de donnée, ces structures peuvent intéragir selon certaines conditions définits par le programmeur : les scripts.
