1.5 Langage informatique

Les languages informatiques existent depuis les tous premiers ordinateurs. Ce qui différencie chaque langage est en premier lieu leur synthaxe (les séparations d'actions, les fonctions, ect...). Cependant la vraie différence viens de leur compilation, les langages peuvent proposer des fonctions de base et le programme compilé peut, selon notre utilisation du langage, être plus au moins optimisé. Ainsi les langages peuvent se spécialiser dans des domaines précis. On utilise par exemple les langages c (c, c++ et c#) pour des applications sur ordinateur ou l'assembleur pour des systémes embarqués.

Les types de langages

Les langages peuvent être divisés en deux catégories : les langage structurés et les langages objets. Leur différence vient de la manière dont un script est construit. En langage structurés, les actions s'executent les une derrière les autres dans un seul script.
Dans le langage objet, le programme est fragmenté en objet qui sont des structures de donnée, ces structures peuvent intéragir selon certaines conditions définits par le programmeur : les scripts.

Les langages notables

En programmation il existe énormement de langages mais certains se démarquent par leur usages, leur popularité ou leur ancienneté.

  • Les langages populaires

    • Les langages C (C,C++,C#). A l'exception du C, se sont des langages objets connu pour être les premiers langages des machines modernes. Ils ont l'avantage d'intéragir avec le systéme de très proche. Ce qui permet d'avoir des applications extrêment rapides. On les retrouvve dans presque tout les domaines : systémes embarqués, logiciels, jeux vidéos, systémes d'exploitation, ect...
    • Le python. Il un successeur directeur du C++, lui permettant approche hybride entre langage structuré et objet. Malgré sa plus grande lenteur que le C++, on le retrouve dans le domaine de l'intelligence artificiel et très rarement dans des logiciels.
    • Le javascript. Il est utilisé pour faire des éléments intéractif en html.
    • Le java. A ne pas confondre avec le javascript, cet un langage objet originellement prévu pour le web. Sa portabilité lui permis aussi de réaliser des logiciels mais aussi des systémes embarqués.
    • L'assembleur. On peut dire que c'est une traduction direct du langage machine (d'instructions binaires à instructions humaine). Malgré la liberté totale qu'il offre au programmeur sur son logiciel, à cause de sa complexité (il faut gérer la mémoire pour la moindre action) et du nombre important de connaissances nécessaire, on lui préfére plutôt le c qui par sa synthaxe rend le code plus simple à lire.

  • Les langages spécialisés

    Les langages de programmation ne sont pas juste une manière de lire et écrire le langage machine. C'est surtout une manière d'optimiser son programme. Par exemple, le javascript n'est pas adapté pour un systéme embarqué à cause de son interprétation des valeurs numériques parfois douteuse. Ce qui est plutôt mauvais si on veut faire un satellite.

    • Les langages scientifiques (pour la computation de nombre) :
      • Python
      • Rust
    • Les langages pour les logiciels :
      • C++ et C
      • Java
      • Assembleur (x86)
    • Les langages éducatifs :
      • Scratch
      • BASIC
    • Les langages pour systémes embarqués
      • Assembleur (z80)
      • C et C++
      • Java
      • Arduino
